# BoredBot

A simple web app that suggests fun activities when you're bored using the [Bored API](https://www.boredapi.com/).

## Features

- Get random activity suggestions
- Filter activities by type
- View activity details (type, participants, price range)
- Clean and intuitive user interface

## Getting Started

### Prerequisites

- Node.js and npm installed

### Installation

1. Install dependencies:
```bash
npm install

 - Install the dependencies and run the project

npm install
npm start
```

Head over to https://vitejs.dev/ to learn more about configuring vite
